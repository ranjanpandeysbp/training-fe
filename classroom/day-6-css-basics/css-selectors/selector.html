Selectors helps us to target/query a specific element/tag from the DOM(document object model) Tree

Types of CSS Selectors

  • ID selector - Give any element/tag and id and the use this id in css file with # to target element
  • Class selector - Give any element/tag and a class and the use this class in css file with . to target element
  • Element/Tag Name selector - Directly use the element/tag-name in css file and start styling

ID Selector Example

I am a P tag with id="example-one-id", I can be accessed in CSS file like this #example-one-id
